1. Project Title / Headline
   # Netflix Content Analytics Dashboard

2.Short Description

An interactive Netflix Content Analytics Dashboard built using [Microsoft Power BI] to analyze global Netflix movies and TV shows. The dashboard provides insights into content trends, genre popularity, country-wise distribution, ratings, duration analysis, and murder mystery content patterns using cleaned and transformed Netflix dataset from [Kaggle]



3. Tech Stack Used

* [Microsoft Power BI]
* Power Query
* DAX (Data Analysis Expressions)
* Data Visualization
* Data Cleaning & Transformation
* Excel / CSV Dataset



4. Data Source

Dataset sourced from:

[Kaggle Netflix Dataset]


5. Features

* Interactive KPI Cards

  * Total Titles
  * Movies Count
  * TV Shows Count
  * Average Duration
  * Total Countries

* Movies vs TV Shows Analysis

* Content Growth Trend by Release Year

* Top 10 Countries Producing Netflix Content

* Genre Popularity Analysis

* Ratings Distribution Analysis

* Duration Distribution & Runtime Analysis

* Director & Cast Insights

* Murder Mystery Content Analysis

* Interactive Slicers & Filters

  * Country
  * Genre
  * Type
  * Rating
  * Release Year

* Dynamic Tooltips & Drill-down Analysis

* Netflix-inspired Dark Theme Dashboard Design


# Key Insights

* Netflix content significantly increased after 2018.
* Drama and Comedy are among the most dominant genres.
* The United States contributes the highest number of titles.
* Most Netflix movies fall within the 90–120 minute duration range.
* Mature audience content (TV-MA) dominates the platform.
